Smith Chart: File Menu

Import Data

This command plots data from a "Gamma file".

The number of data points that can be imported is only limited by the file size which is usually 64K Bytes. QuickSmith does not store this data in its memory thus a redraw command erases the trace.

 

Export Data

This command exports the swept trace from the smith chart to a "Gamma file".

 

Edit Data

This command opens a text editor which may be used to view or edit ASCII files.

 

Print Chart

Use this command to print the Smith Chart.

 

Exit

This command ends QuickSmith.
